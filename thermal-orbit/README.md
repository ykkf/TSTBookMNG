# Manga Manager (Prototype)

マンガの購入管理を行うためのWebアプリケーション（PWA対応）です。
スマートフォンでの利用を想定しており、ホーム画面に追加することでアプリのように動作します。

## 機能
- **未購入リスト**: 買い忘れ防止のためのリスト管理
- **購入済みリスト**: 重複購入防止のための履歴管理
- **出版社管理**: 出版社の追加・削除
- **検索・絞り込み**: タイトルや出版社でのフィルタリング
- **自動削除**: 購入済みデータの自動整理（期間設定可）
- **PWA対応**: オフライン動作、ホーム画面への追加

## 使い方
1. **未購入リストに追加**:
   - 右下のカメラアイコン（または手動入力）からマンガを登録します。
2. **購入済みにする**:
   - リストのチェックボックスをタップすると「購入済み」に移動します。
3. **設定**:
   - 右上の歯車アイコンから、出版社の管理やデータの自動削除設定が行えます。

## インストール方法（家族と共有）
詳しい手順は [deployment_guide.md](./deployment_guide.md) をご覧ください。
簡単に言うと、NetlifyなどにアップロードしてURLを家族に送り、「ホーム画面に追加」してもらうだけで使えます。

## データについて
データは各端末のブラウザ（LocalStorage）に保存されます。
機種変更時やブラウザのキャッシュクリアでデータが消える可能性があるためご注意ください。
現在のバージョンでは、家族間でのデータ同期は行われません（個人の管理用です）。
